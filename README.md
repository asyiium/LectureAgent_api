### Работа с приложением
Если хотите локально запустить и посмотреть, что и как делает код, вам потребуются собственные API ключи GigaChat и SaluteSpeech, а также наличие файла сертификатов минцифры (скачиваете с сайта Госуслуг, закидывайте в любую директорию и записываете путь в .env).

Все это + пути и пароли указываются через файл с переменными окружения .env. Возможный формат может выглядеть так:
```
# database access
REDIS_HOST=redis

REDIS_PASSWORD=...
REDIS_USER=...
REDIS_USER_PASSWORD=...
REDIS_SESSION_TTL=...
REDIS_PORT=...
 
VECTOR_STORE_PATH=...

# api keys
AUDIO_LLM_API_KEY=...
TEXT_LLM_API_KEY=...

# certificate
CA_BUNDLE_PATH=...
```
